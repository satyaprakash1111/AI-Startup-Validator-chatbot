# AI Startup Validator Chatbot

## 🚀 Overview

AI Startup Validator Chatbot is an AI-powered web application that helps entrepreneurs evaluate startup ideas based on innovation, feasibility, market demand, competition, and scalability.

## ✨ Features

- AI-powered startup idea evaluation
- Startup scorecard
- Interactive chatbot
- Save startup ideas
- Detailed AI feedback

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Node.js
- Google Gemini API

## 🚀 Run Locally

```bash
npm install
npm run dev
```
